# Changelog

## v0.3
- 新增系统主题自适应 + 手动切换：跟随系统亮/暗色自动切换，支持顶部太阳/月亮按钮覆盖，并全局使用 CSS 变量统一配色。
- 顶部信息条重构：GitHub 风格灰/蓝分割胶囊（产品名+版本、TS 5.9、React 19），尺寸与间距缩小，随滚动悬停在页面顶端，打开媒体弹窗时自动隐藏。
- 布局对齐：顶部条宽度与主体区域一致，去除多余空白，滚动到顶不再出现留白。

## v0.2
- 账号列表/媒体面板使用滑块式互斥切换：媒体类型（图片/视频）、排序（时间/名称）与时间升降序（时间+/时间-）。
- 预览弹窗体验强化：键盘左右/ESC、点击图片关闭、左右按钮、滑动、滚轮左右切换，开启时禁止背景滚动。
- 浮动回顶按钮对齐内容区域；列表高度随视口伸展。
- 修复分类目录计数受预览条数限制导致的统计缺失，提升缓存版本以刷新数据。
- 账号过滤恢复为“当前类型数量>0”策略。

## v0.1
- 初版交付：Fastify + TypeScript 后端扫描媒体目录，提供 `/api/folder` 和 `/media/*`；Vite + React 前端浏览、预览媒体；`run-dev.sh` 一键启动前后端。
