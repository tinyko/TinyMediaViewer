# Changelog

## v0.5.0
- 交互特效：独角兽/心形光标，心跳线框 + 同步粒子爆点，卡片边缘随心跳随机色高亮；心跳粒子量提升，避免 hover 卡顿（仅首次启动定时器）。
- 兼容性：构建目标面向 Safari 14+，增加 flex gap 回退，iPad 触控/鼠标均可触发粒子。
- 视觉：心形/卡片高亮色跟随每次心跳随机 hue；光标/心跳/粒子坐标统一，对齐偏移清零。

## v0.4.1
- 更新网页标签与 favicon，统一品牌为 Tiny Media Viewer。

## v0.4
- 品牌更新为 Tiny Media Viewer，顶部胶囊沿用 GitHub 风格并同步版本号。
- 布局调整：主页面高度锁定，左侧列表与媒体区独立滚动，顶部胶囊和媒体工具条固定；回到顶部按钮随媒体区滚动。
- 媒体工具条精简：移除账号名，仅保留媒体计数胶囊（灰底+按占比渐变填充）与微型时间排序开关。
- 其他：媒体网格滚动体验修复，媒体工具条/切换按钮缩小以贴合顶部行高度。

## v0.3
- 新增系统主题自适应 + 手动切换：跟随系统亮/暗色自动切换，支持顶部太阳/月亮按钮覆盖，并全局使用 CSS 变量统一配色。
- 顶部信息条重构：GitHub 风格灰/蓝分割胶囊（产品名+版本、TS 5.9、React 19），尺寸与间距缩小，随滚动悬停在页面顶端，打开媒体弹窗时自动隐藏。
- 布局对齐：顶部条宽度与主体区域一致，去除多余空白，滚动到顶不再出现留白。

## v0.2
- 账号列表/媒体面板使用滑块式互斥切换：媒体类型（图片/视频）、排序（时间/名称）与时间升降序（时间+/时间-）。
- 预览弹窗体验强化：键盘左右/ESC、点击图片关闭、左右按钮、滑动、滚轮左右切换，开启时禁止背景滚动。
- 浮动回顶按钮对齐内容区域；列表高度随视口伸展。
- 修复分类目录计数受预览条数限制导致的统计缺失，提升缓存版本以刷新数据。
- 账号过滤恢复为“当前类型数量>0”策略。

## v0.1
- 初版交付：Fastify + TypeScript 后端扫描媒体目录，提供 `/api/folder` 和 `/media/*`；Vite + React 前端浏览、预览媒体；`run-dev.sh` 一键启动前后端。
